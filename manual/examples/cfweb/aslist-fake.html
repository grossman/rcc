This is a very trimmed down example of the Web interface to the control flow analyzer. The example is static HTML, but the real analyzer uses a DB backend.

The "AS view" summarizes the number of sessions per remote AS. Clicking on the number of sessions provides details about each one of those sessions. The real control flow analyzer also has a "router view", which shows BGP sessions by router. We do not show this functionality in this online example.
